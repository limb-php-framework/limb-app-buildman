Datasource Example


This is an example using the datasource tag and the import tag to illustrate how datasource variable scoping works inside of a template file.

{$Animal} = {$Animal} We cant use {$^Animal} here since no datasource upper than root datasource {$#Animal} = {$#Animal}
{$Food} = {$Food} We cant use {$^Food} here since no datasource upper than root datasource {$#Food} = {$#Food}

This is inside a datasource tag
{$Animal} = {$Animal} {$^Animal} = {$^Animal} {$#Animal} = {$#Animal}
{$Food} = {$Food} {$^Food} = {$^Food} {$#Food} = {$#Food}
This is inside a nested datasource tag
{$Animal} = {$Animal} {$^Animal} = {$^Animal} {$^^Animal} = {$^^Animal} {$#Animal} = {$#Animal}
{$Food} = {$Food} {$^Food} = {$^Food} {$^^Food} = {$^^Food} {$#Food} = {$#Food}
This is inside a nested datasource tag but data taken from root datasource using from attribute
{$Animal} = {$Animal}
{$Food} = {$Food}
