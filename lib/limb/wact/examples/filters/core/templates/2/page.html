Core filters Example

Description:

This page demonstrates how to use several core filters such as:
  • const
  • url
  • hex
  • notags
  • etc.

Const filter

{$'MY_TESTING_TEMPLAE_EXAMPLE_CONSTANT'|const} = {$'MY_TESTING_TEMPLAE_EXAMPLE_CONSTANT'|const}

Url filter

{$my_url|url} = {$my_url|url} (real value is {$my_url})

Notags filter

{$my_text|notags} = {$my_text|notags} (real value is {$my_text})

Hex filter

{$my_hex|hex} = {$my_hex|hex} (real value is {$my_hex})
