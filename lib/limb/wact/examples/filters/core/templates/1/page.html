Text processing filters Example

Description:

This page demonstrates how to use several text processing filters such as:
  • upper
  • lower
  • trim
  • nl2br
  • number
  • capitalize
  • etc.
It also shows how to combine several filters at time.

Uppercase filter

{$'Text'|uppercase} = {$'Text'|uppercase}

Lowercase filter

{$'Text'|lowercase} = {$'Text'|lowercase}

Trim filter

{$' Text '|trim} = {$' Text '|trim}
{$'&& Text &&'|trim:'&'} = {$'&& Text &&'|trim:'&'}

Many filters at time

{$'&& Text &&'|trim:'&'|uppercase} = {$'&& Text &&'|trim:'&'|uppercase}

Capitalize filter

{$'text'|capitalize} = {$'text'|capitalize}

Number filter

{$1234|number} = {$1234|number}
{$1234|number:'2' ,'.'} = {$1234|number:'2' ,'.'}
{$1234|number:'2' ,'.', '-'} = {$1234|number:'2' ,'.', '-'}

Wordwrap with NL2BR filter

{$'some very long text'|wordwrap:10|nl2br} = {$'some very long text'|wordwrap:10|nl2br}

{$'some very long text'|wordwrap:10|nl2br|raw} (no htmlspecialchars by default) = {$'some very long text'|wordwrap:10|nl2br|raw}
