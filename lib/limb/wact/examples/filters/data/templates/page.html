Data Filters Example

Description:

This page demonstrates how to use several data filters such as:
  • clip
  • math
  • stats

Clip filter

{$'Very long text'|clip:10} = {$'Very long text'|clip:10}
{$'Very long text'|clip:10,5} = {$'Very long text'|clip:10,5}
{$'Very long text'|clip:10,1,"..."} = {$'Very long text'|clip:10,1,"..."}
{$'Very long text'|clip:10,1,"...","y"} = {$'Very long text'|clip:10,1,"...", "y"}

Math filter

{$'1 + 2'|math} = {$'1 + 2'|math}
{$'2 * 2'|math} = {$'2 * 2'|math}
